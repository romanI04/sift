How it works
01

Index at build time

Run sift index ./docs to chunk your content and compute embeddings. Output is a single SQLite file.

02

Drop in the widget

Add <sift-search> to your HTML. 94KB bundle, loads on demand. No build step required.

03

Search runs client-side

Keywords match instantly via FTS5. The ML model loads in a WebWorker and upgrades results to semantic.

Install in 30 seconds
# index your docs
pip install sift-search
sift index ./docs -o public/sift-index.db

# add to your site
<sift-search db="/sift-index.db"></sift-search>
